# 아이콘 준비 방법

## Windows 설치 파일용 icon.ico

icon.svg를 ico로 변환:
1. https://convertio.co/svg-ico/ 접속
2. icon.svg 업로드
3. 256x256 크기로 변환
4. 이 폴더에 icon.ico 저장

또는 온라인 도구:
- https://favicon.io/favicon-converter/
- https://icoconvert.com/

## 크기 요구사항
- Windows .ico: 256x256 (최소 64x64)
- macOS .icns: 1024x1024
